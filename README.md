# 🍏 Ecommerce App

**Minimalist. Premium. Fast.**

Ecommerce App is a high-fidelity marketplace built with Flutter. It features an "Apple Store" inspired aesthetic, focusing on clean typography, high-resolution product imagery, and a seamless checkout experience.

## ✨ Features

- 💎 **Silent Luxury Design** — A high-end minimalist interface optimized for premium product discovery.
- 🛍️ **Smart Catalog** — High-performance grid rendering with lazy-loading and advanced filtering.
- 💳 **Seamless Checkout** — A refined, multi-step checkout process designed for high conversion.
- 📱 **Adaptive Layout** — Optimized for all mobile screen sizes with fluid responsiveness.
- ⚡ **Clean Architecture** — Scalable codebase built using Riverpod and modular domain design.

## 🛠️ Tech Stack

- **Framework**: Flutter
- **State Management**: Riverpod
- **Design System**: Custom Minimalist (Aurex One)
- **Architecture**: Clean Architecture
